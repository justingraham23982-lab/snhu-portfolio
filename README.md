# CS 210 Portfolio Project – Chada Tech Clock

## 1. Project Summary
This project created a dual 12-hour and 24-hour clock display for Chada Tech. The program continuously updates the time, formats it correctly, and allows the user to adjust hours, minutes, and seconds. The goal was to demonstrate time manipulation, input handling, and clean C++ program structure.

## 2. What I Did Well
I did well in organizing the program into clear functions that each handled one task, such as updating the time and formatting the output. My logic for incrementing seconds, minutes, and hours was accurate and avoided rollover errors.

## 3. Where I Could Enhance My Code
I could improve the program by adding input validation and reducing repeated code in the time‑update logic. These changes would make the program more efficient, more secure, and easier to maintain. I could also separate the display formatting into its own class for better modularity.

## 4. Challenges and How I Overcame Them
The most challenging part was handling the time rollover correctly, especially switching between 12‑hour and 24‑hour formats. I overcame this by breaking the logic into small steps and testing each part. I also used online documentation and examples to confirm correct formatting rules.

## 5. Transferable Skills
This project strengthened my skills in C++ functions, loops, conditionals, and modular program design. These skills transfer directly to future programming courses and any project that requires structured logic and clean code organization.

## 6. Maintainability, Readability, and Adaptability
I made the program maintainable by using clear function names, consistent formatting, and comments explaining the time logic. The structure makes it easy to update or expand the program, such as adding alarms or additional time zones.
